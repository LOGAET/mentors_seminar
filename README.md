## Описание проекта

Проект состоит из двух независимых микросервисов:

### TODO Service
Микросервис для управления задачами с полным набором CRUD операций:
- Создание задач
- Получение списка задач
- Получение задачи по ID
- Обновление задачи
- Удаление задачи

### ShortURL Service
Микросервис для сокращения длинных URL-адресов:
- Создание короткой ссылки из полного URL
- Перенаправление по короткой ссылке
- Получение статистики по короткой ссылке

## Архитектура

### Диаграмма архитектуры системы

- *Два микросервиса (TODO Service и ShortURL Service)*
- *Docker контейнеры с портами 8000 и 8001*
- *Docker Volumes для хранения данных (todo_data и shorturl_data)*
- *SQLite базы данных внутри контейнеров*
- *Взаимодействие клиента с сервисами через HTTP API*

```
┌─────────────┐         ┌─────────────┐
│   Клиент    │────────▶│ TODO Service│
│  (Browser)  │         │  :8000      │
└─────────────┘         └─────────────┘
                               │
                               ▼
                        ┌─────────────┐
                        │  SQLite DB  │
                        │  todo.db    │
                        └─────────────┘

┌─────────────┐         ┌─────────────┐
│   Клиент    │────────▶│ShortURL     │
│  (Browser)  │         │ Service     │
└─────────────┘         │  :8001      │
                               │
                               ▼
                        ┌─────────────┐
                        │  SQLite DB  │
                        │  urls.db    │
                        └─────────────┘
```


## Технологический стек

- **FastAPI** - современный веб-фреймворк для создания API
- **SQLite** - легковесная база данных
- **Uvicorn** - ASGI сервер для запуска FastAPI приложений
- **Docker** - контейнеризация приложений
- **Python 3.11** - язык программирования

## Установка и запуск

```
docker run -d -p 8000:80 -v todo_data:/app/data --name todo-service darkhatoon/todo-service:latest
docker run -d -p 8001:80 -v shorturl_data:/app/data --name shorturl-service darkhatoon/shorturl-service:latest
```

## API Документация

### TODO Service API

Базовый URL: `http://localhost:8000`

#### Эндпоинты:

- `POST /items` - Создать новую задачу
- `GET /items` - Получить список всех задач
- `GET /items/{item_id}` - Получить задачу по ID
- `PUT /items/{item_id}` - Обновить задачу
- `DELETE /items/{item_id}` - Удалить задачу
- `GET /docs` - Документация

#### Модель данных Item:

```json
{
  "id": 1,
  "title": "Название задачи",
  "description": "Описание задачи",
  "completed": false
}
```

### ShortURL Service API

Базовый URL: `http://localhost:8001`

#### Эндпоинты:

- `POST /shorten` - Создать короткую ссылку
- `GET /{short_id}` - Перенаправить на полный URL
- `GET /stats/{short_id}` - Получить статистику по короткой ссылке
- `GET /docs` - Документация

#### Модель данных:

```json
{
  "url": "https://example.com/very/long/url/path"
}
```
